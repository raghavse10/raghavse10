

Code Set-Up:
__________________________________________

This assignment is made to run on localhost MongoDB, Python FastAPI, and React.js.

Database:
1. Please connect your MongoDB with localhost connection string "mongodb://localhost:27017/" and create a database named "DBVN", having one collection 
named "connections".
2. Please import the JSON named "DBVN.connections.json" provided in the .zip, this is the same excel data converted to JSON, to easily import into MongoDB.

Backend:
1. The backend code is provided in the folder named "backend" in the .zip. After installing FastAPI, run the backend using "uvicorn main:app --reload" inside backend folder.
It should be on port 8000

Frontend:
2. Both Frontend source code as well as build is provided without the node_modules.
